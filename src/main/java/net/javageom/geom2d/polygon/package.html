Definitions and implementations of shapes composed several line segments: polylines, rings, polygons. Contains several polygon implementations, to manage the different types of polygons.

Package organization:

  • Polygons:
    • Polygon2D: main interface for all polygonal shapes
    • SimplePolygon2D: implementation for simply connected polygons
    • MultiPolygon2D: implementation for polygons composed of several pieces
  • Polylines:
    • LinearCurve2D: abstract class for any type of polyline
    • Polyline2D: implementation for open polyline
    • LinearRing2D: implementation for closed polylines, typically polygon boundaries
  • Utility class:
    • Polygons2D: utilities for polygons
    • Polylines2D: utilities for polylines
